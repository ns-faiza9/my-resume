# Shaik Naseera Faiza - Resume

My personal resume page, hosted on GitHub Pages.

## How to Deploy

1. **Add your photo** — Place `2500032104.jpeg` (or rename it) in this folder. The HTML already references `2500032104.jpeg`.

2. **Push to GitHub** (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial resume page"
   git remote add origin https://github.com/ns-faiza9/ns-faiza9.github.io.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Go to your repo on GitHub → **Settings** → **Pages**
   - Under "Branch", select `main` and `/ (root)`
   - Save → Your site will be live at `https://ns-faiza9.github.io/`
